# fast-api-training
Files created as part of training https://fastapidozero.dunossauro.com/

# Running the application:

> poetry install

> Start the application locally (http://127.0.0.1:8000)
>> poetry run fastapi dev .\fast_zero\app.py

> Start the application in the network
>> poetry run fastapi dev .\fast_zero\app.py --host 0.0.0.0
